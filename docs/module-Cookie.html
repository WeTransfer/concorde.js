(inner) configure(options)

Source:
  • cookie/src/cookie.js, line 41
Since:
  • 1.0.0
Configure default options used when settings values for cookies. If all your cookies must present the same property, like secure, use this method before setting the cookie values.
Example
import Cookie from '@wetransfer/concorde-cookie';

Cookie.configure({ secure: true });
Cookie.set('foo', 'bar');
// => foo=bar; secure'
Parameters:
Name Type Description
options Object Default options for future cookie values.
Properties
Name Type Description
expires Date Cookie expiration date
path String Sets the URI to which the Cookie applies. Must be an absolute path.
domain String Sets the domain to which the Cookie applies. If a domain is specified, subdomains are always included.
secure boolean Cookie to only be transmitted over secure protocol as https.

(inner) get(key, optionsopt)

Source:
  • cookie/src/cookie.js, line 64
Since:
  • 1.0.0
Get the value from a cookie, given a key.
Example
import Cookie from '@wetransfer/concorde-cookie';

Cookie.get('foo');
// => 'bar'

Cookie.get('nonexistant', { defaultValue: 1 });
// => 1

Cookie.get('yolo', { raw: true });
// => 'bar%20yolo'
Parameters:
Name Type Attributes Description
key String Cookie key name.
options Object <optional>
Properties
Name Type Attributes Default Description
defaultValue Any <optional>
 null Default value in case cookies are not available or the value has not been set previously.
raw boolean <optional>
 false If true, the value of the cookie is not decoded.
