(inner) isOutdated(supportedBrowsers) → {boolean}

Source:
  • browser.js, line 196
Since:
  • 1.0.0
Determines if the actual browser is not part of the provided browser or list of browsers.
Example
import Browser from '@wetransfer/concorde-browser';

Browser.isOutdated([
  'explorer >= 9.0',
  'chrome >= 43',
  'firefox >= 42',
  'safari >= 5'
]);
// => false

Browser.isOutdated('explorer < 6');
// => true
Parameters:
Name Type Description
supportedBrowsers Array | String A browser or list of supported browsers, with version check condition.
Returns:
True if the actual browser does not match some of the browser on the list.
Type
boolean
